# 闪耀星骑士 Android 动画替换 MOD

这是 Android 版 Release 0.1.2。安装后，支持角色发动“通常攻击 1”和“通常攻击 2”时出现的角色演出都会替换为该角色的高画质成人变身动画（R18），战斗和非战斗动画预览均支持。实际攻击动作、伤害和战斗速度不会改变。

后续可能开发仅在“通常攻击 2”时生效的可选版本。

## 当前不支持的角色

露露（`[炎宿せし宝石] ルルゥ`，内部 ID `1141001`）当前不会应用 MOD，通常攻击时继续使用游戏原始演出。

## 使用条件

- Windows 电脑和 USB 数据线。
- 手机已开启 USB 调试并允许当前电脑连接，无需 Root。
- 已安装安卓版兼容安装包（APK），包名为 `jp.co.fanzagames.twinklestarknightsx_a_mod`。请前往 [anosu/DMM-Mod](https://github.com/anosu/DMM-Mod) 获取该安装包。
- 首次安装需要联网。实际下载量取决于手机已有缓存；2026 年 7 月测试约需 0.66 GiB。

## 开启 USB 调试

1. 打开手机“设置 → 关于手机 → 版本信息”。
2. 连续点击“软件版本号”约 7 次，根据提示输入锁屏密码，开启开发者模式。
3. 返回设置，进入“系统管理与升级 → 开发者选项”。部分品牌会将其放在“更多设置”中。
4. 开启“USB 调试”。连接电脑后保持手机解锁，并在授权弹窗中选择允许。

如果找不到对应入口，可在设置中搜索“版本信息”“开发者选项”或“USB 调试”。

## 安装

1. 解压整个 Release ZIP，不要只单独取出 BAT 文件。
2. 确认手机已经安装并能正常启动安卓版兼容安装包，然后完全退出游戏。
3. 用 USB 数据线连接手机和电脑，保持手机解锁。
4. 手机上出现“允许 USB 调试”时，勾选“始终允许使用这台计算机”，再点击“允许”。这是手机端必须完成的确认。
5. 在电脑上打开解压后的文件夹，双击 `Apply-TskSkinSwap-Android.bat`。不要关闭弹出的黑色窗口，也不要断开手机。
6. 首次运行会自动准备工具并下载缺失资源，请等待电脑窗口显示 `Installation completed`。
7. 安装器随后会在手机上自动启动《闪耀星骑士》。**只有电脑显示安装完成，并且手机上的游戏已经被自动拉起，才表示本次应用完成。**

如果手机没有弹出 USB 调试授权，请关闭后重新开启 USB 调试，再拔插一次数据线。若电脑显示失败或手机上的游戏没有自动启动，请保持窗口内容不变，先按上方错误提示处理，再重新运行 BAT。

已经使用 Android 旧版的用户无需卸载：下载并完整解压 0.1.2，直接重新运行新的 BAT 即可。手机上仍然有效的缓存和已下载资源会继续复用。

安装器只读取当前资源目录，优先直接复用手机 UnityCache，仅从游戏官方 CDN 下载缺失的变身 Bundle。它不会复制手机上已有的数 GB 游戏数据，也不会修改游戏原始 Bundle。

## 游戏更新后

先正常启动游戏一次，让客户端完成资源更新，再重新双击安装 BAT。仍然有效的缓存和 MOD 资源会直接复用，只补充发生变化的文件。

## 卸载

双击 `Uninstall-TskSkinSwap-Android.bat`。默认保留已下载资源，便于以后重新安装。

若确定要同时释放手机空间，可连接手机后在 PowerShell 中运行：

```powershell
.\Uninstall-TskSkinSwap-Android.ps1 -RemoveBundles
```

## 常见问题

### 找不到设备

确认 USB 调试已开启、手机保持解锁，并在手机上允许这台电脑进行调试。更换 USB 连接模式或数据线后重试。

### 提示安卓版兼容安装包不兼容

当前版本不会自动将原版安装包改为兼容版本。请先前往 [anosu/DMM-Mod](https://github.com/anosu/DMM-Mod) 获取并安装安卓版兼容安装包。

### 下载中断

重新运行 BAT。未完成文件会断点续传，已经验证并推送的文件不会重复下载。

## 资源说明

Release 不包含安卓版兼容安装包或游戏动画。所需资源会根据当前游戏版本下载并保存在用户自己的电脑和手机上，请勿重新分发。
